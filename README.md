# `countinggame` README

## 项目简介

countinggame.c 实现“从 1 开始报数，反转方向、m 号玩家拍掌计数”的模拟：

- 当 `m` 号玩家报数满足“7倍数或包含7”时，计数一次；
- 当拍掌次数达 `k` 时，输出当前报数。

---

## 输入行为

1. 输入提示：
   - `2<=n<=100`
   - `1<=m<=n`
   - `1<=k<=100`
2. 最多 10 组数据；
3. 每组：`n m k`
4. 输入 `0 0 0` 结束；
5. 数据不合规时：
   - 显示 `输入数据不合法，请重新输入：`
   - 重新读该组（`i--` 处理）。

---

## 变量说明

- `n[10], m[10], k[10]`：存最多 10 组输入；
- `player[101]`：玩家报数缓存；
- `num`：当前报数；
- `j`：当前玩家编号；
- `operation`：方向符号 `+1`/`-1`；
- `count`：m 号玩家拍掌累计。

---

## 核心流程

1. 对每个 `i`：
   - `num=1, count=0, j=1, operation='-'` 开始（初始首位遇端点时先翻为 `+`）。
2. 循环：
   - 若 `j==1` 或 `j==n`，`operation = -operation`；
   - `player[j] = num`；
   - 当 `j==m`：
     - `num % 7 == 0` 或 `num` 含 `7` → `count++`；
   - 若 `count == k`：`printf("%d\n", num)`，结束当前组；
   - 否则：`j = j+operation`，`num++`。
3. 下组继续。

---

## 输出结果

每组输出一行：满足 `m` 号玩家第 `k` 次拍掌时的 `num`。

---

## 使用示例

输入：

```
5 2 1
8 3 2
0 0 0
```

输出：

```
7
21
```

---

## 编译运行

```bash
gcc countinggame.c -o countinggame
./countinggame
```
